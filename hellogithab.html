С помощью Git программисты и разработчики ориентируются в коде и отслеживают изменения ГИТ помогает вернуть файлы в исходное состояние и видеть изменеия, внесенные в определенный период. Разработчик выполняет разные команды (например коммит, пуш), а все изменения синхронизируются с центральным репозиторием. ГИТ - это система контроля версий, а ГИТХАБ - онлайнсервис, по сути социальная сеть. Одна из основных целей ГИТХАБ - быть единым местом для проектов с исходным кодом. Предпологается, что пользователь делится чем-то полезным, а другие люди смогут учавствовать в разработке.
